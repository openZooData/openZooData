# Contributing to OpenZooData

Thank you for contributing to OpenZooData.

## Goals

We welcome contributions that improve:

- Open data quality
- Federation support
- API stability
- Documentation
- Accessibility
- Internationalization

## Pull Requests

Please:

1. Fork the repository
2. Create a feature branch
3. Keep commits clean and understandable
4. Open a pull request with a clear description

## Coding Standards

- Prefer readable code over clever code
- Document APIs and database changes
- Never commit secrets or API keys
- Keep compatibility in mind

## Data Quality

Zoo and species data should:
- Reference Wikidata whenever possible
- Use scientific names consistently
- Avoid proprietary or copyrighted datasets

## Community

Please remain respectful and constructive.
