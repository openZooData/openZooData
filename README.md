# OpenZooData

OpenZooData is an open infrastructure project for structured zoo, animal, enclosure and visitor information.

## Vision

OpenZooData aims to provide:

- Open zoo data
- Public APIs
- Feed-based synchronization
- Structured species and enclosure datasets
- Wikidata integration
- Offline-capable datasets
- Community contributions
- Open standards for zoo information systems

## Core Principles

- Open infrastructure
- Publicly accessible data
- Federation instead of vendor lock-in
- Reusable APIs and feeds
- Compatibility with Wikidata and Open Data ecosystems
- Separation of open infrastructure and proprietary client applications

## Components

### Open Source Components

- OpenZooData Server
- Public API
- Feed/Manifest infrastructure
- Database schema
- Open data formats

### Proprietary Components

- ZooGuide iOS App
- ZooCreator
- Analytics dashboards
- Commercial integrations
