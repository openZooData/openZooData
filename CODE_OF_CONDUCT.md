# Code of Conduct

## Our Pledge

We are committed to creating an open and welcoming environment for everyone.

## Expected Behavior

- Be respectful
- Be constructive
- Be inclusive
- Help others learn

## Unacceptable Behavior

- Harassment
- Discrimination
- Personal attacks
- Publishing private information

## Enforcement

Project maintainers may remove comments, contributions or users that violate this code of conduct.
