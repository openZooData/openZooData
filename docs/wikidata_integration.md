# Wikidata Integration

OpenZooData is designed for close interoperability with Wikidata.

## Species References

Species entries currently support:

- Wikidata IDs
- scientific names
- IUCN status IDs
- GBIF taxon keys

## Existing Tooling

The repository already contains tooling for:

- Wikidata synchronization
- translation checks
- taxonomy validation
- species enrichment
- missing translation detection

## Strategic Goal

OpenZooData aims to:

- consume structured open data
- improve zoo-related datasets
- contribute data back into the ecosystem
- support multilingual species information
